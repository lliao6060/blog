# ngrok設定

# tags: ngrok

ngrok是個好東西，方便讓你把做好的網站demo給人家看

# 基本安裝

  1. 安裝ngrokopen in new window
  2. 在ngrok終端機中設定token
 ngrok authtoken `your token`
1
  1. 在ngrok終端機中開啟你的port
ngrok http --region ap --host-header=rewrite `your port`
1

# 壞掉的話

  1. 用Chocolateyopen in new window安裝：choco install ngrok
  2. 打開powershell以系統管理員身分執行並輸入
Set-ExecutionPolicy Bypass -Scope Process -Force; [System.Net.ServicePointManager]::SecurityProtocol = [System.Net.ServicePointManager]::SecurityProtocol -bor 3072; iex ((New-Object System.Net.WebClient).DownloadString('https://chocolatey.org/install.ps1'))
1

就可以開始用了!!

# 使用方法

  1. 從package商店open in new window搜尋你要的套件然後複製到powershell
  2. 一直按確定然後按Y，然後以前ngㄟok怎麼用就繼續怎麼用
