# 環境建立

# tags: 上工前環境

  1. 安裝IDE(編輯器) VS CODE

  2. 安裝GIT(windows需要)

  3. 設定IDE(擴充功能)

    • Beautify
    • Chinese (Traditional) Language Pack for Visual Studio Code
    • Color Info 可以不用透過F12更改顏色
    • HTML Boilerplate 可以設定你的html模板 一鍵產生open in new window
    • HTML CSS Support
    • htmltagwrap 可以多行選取外層加tag(Press Alt + W or Option + W for Mac.)
    • open in browser
    • sass
    • vetur
    • screenify 截圖 (command shift + S (mac))
    • Code Spell Checker 拼字檢查
    • indent rainbow indent彩虹 (vs code內建了)
    • Rainbow Brackets 大中小括弧會有顏色區分 (vs code內建了)
    • 裝ESlint
    • Auto Close Tag 會自動補齊結尾 (vs code內建了)
    • DotENV .env文件高亮
    • [Vue] i18n 神器 i18n-ally(可選)open in new window
    • Auto Rename Tag 自動重命名配對的 HTML / XML 標籤，也可以在 JSX 中使用。
    • Highlight Matching Tag 突出顯示匹配的開始或結束標籤。
    • koroFileHeader 生成文件頭部註釋和函數註釋(可選)open in new window
    • Vue VSCode Snippetsopen in new window
    • Trailing Spaces 自動偵測多餘的空格
    • GitLens 可查看該行代碼是誰寫的
    • Colorize 可以立即查看當前色碼的顏色

  4. 設定Chrome擴充功能

    • AdBlock 擋廣告
    • JSON Viewer 看api資料時比較清楚
    • line
    • google翻譯
    • wappalyzer(可以看到他人網站使用技術)
    • Responsive Viewer 一次查看各尺寸RWD
    • Image Downloader 一鍵下載目標網站的所有圖片(復刻網站很實用
